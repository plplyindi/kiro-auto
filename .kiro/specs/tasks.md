# Kiro 每日自动化任务

这是一个示例任务文件。请根据您的需求修改这些任务。

## 任务 1: 每日问候
创建一个名为 `daily-greeting.txt` 的文件，内容包含：
- 当前日期和时间
- 一句问候语："今天是美好的一天！"

## 任务 2: 系统信息
创建一个名为 `system-info.txt` 的文件，包含：
- 当前运行环境信息
- GitHub Actions 运行编号

## 任务 3: 任务完成标记
创建一个名为 `task-completed.txt` 的文件，内容为：
- "所有任务已完成"
- 完成时间

---

**注意：**
- 这些是示例任务，您可以修改 `.kiro/specs/tasks.md` 来定义自己的任务
- 任务可以是：文件操作、数据处理、API 调用等
- Kiro 会按顺序执行这些任务
